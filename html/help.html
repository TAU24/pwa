Tips

  • All adventures in a square of block size as entered in the settings are loaded.
  • When moving over a significant distance, new adventures are loaded.
  • Moving and zooming the map also does load other adventures.
  • A maximum number of labs, depending on the donation status, can be loaded. When this number of labs in block size is too big the alembic shakes. The globe shakes when the number of labs to be displayed on the map exceeds the limit.
  • Clicking on the cross-hairs wil center the map on the current position. Double-clicking will keep the map centered.
  • The circles on the map show the size of the geofence.
  • This distance shown is the distance to the geofence.
  • The input bar is active when inside the geofence. It's becomes inactive on leaving the geofence. Unless the lab is "red". Meaning wrong answers can be corrected outside the geofence.
  • Adventures can be logged when there is an answer in the input bar.
  • Update interval is the time between two updates of the position. When set to 0 the position is updated when your device detects a change in location.
  • Notif. dist. is the distance from the geofence at which a notification will be triggered. When set to 0, no notifications will be given.
  • Due to limitations of progressive web apps, notifications will only be shown when the app is open.
  • Clicking on the notification will open the corresponding lab.
